# Expedition33Editor
This project will be an Android application that will allow the user to consult information about Expedition33. In addition, it will also allow the user to create the builds (pictos, weapons, etc.) of their characters in the application, or make quick checks on them, like the damge inflicted by maelle using stendhal for example.

The project will be develop in kotlin multiplatform & compose multiplatform and will try to integrate an AI in the builds 

The develop is expected to start soon 2026

The app name is provisional
